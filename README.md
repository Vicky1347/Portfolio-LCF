# Portfolio Management System

A comprehensive portfolio management platform built with React frontend and Spring Boot backend, supporting stocks, cryptocurrencies, commodities, and real estate investments.

## Tech Stack

- **Frontend:** React, Bootstrap
- **Backend:** Spring Boot 3.2.2
- **Database:** MySQL 8.2.0
- **API Integration:** Alpha Vantage (Stock prices)

## Project Structure

```
internship_project/
├── frontend/          # React application
│   ├── src/
│   ├── public/
│   └── package.json
├── backend/           # Spring Boot application
│   ├── src/
│   └── pom.xml
└── databases/         # Database configurations
```

## Getting Started

### Prerequisites
- Node.js 14+
- Java 11+
- MySQL 8.0+

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

### Backend Setup

```bash
cd backend
mvn clean install
mvn spring-boot:run
```
